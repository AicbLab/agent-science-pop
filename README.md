# 多AI智能体协同的科普创新路径研究 —— 仿真实验平台

基于智能体建模(Agent-Based Modeling, ABM)的动态演化仿真系统，用于研究多AI智能体协同创作科普内容的创新机制与优化策略。

## 项目简介

本项目构建了一个多智能体仿真平台，模拟四种不同类型的AI智能体在科普内容创作生态中的协作演化过程。通过对比基线实验与三种政策干预实验，探索提升科普内容质量与系统创新能力的有效路径。

## 智能体类型

| 智能体类型 | 角色描述 | 核心特征 |
|-----------|---------|---------|
| 内容生成智能体 (ContentGenerator) | 创作科普内容 | 较高知识水平、较高创新能力 |
| 知识审核智能体 (KnowledgeReviewer) | 审核内容准确性 | 很高知识水平、较低创新能力 |
| 传播分发智能体 (ContentDistributor) | 分发推广内容 | 中等知识、较高声誉 |
| 受众反馈智能体 (AudienceFeedback) | 提供用户反馈 | 较低知识、较低声誉 |

## 实验设计

### 基线实验
无外部干预的自然演化过程，作为对照基准。

### 实验一：资源定向投入策略
对内容生成智能体注入额外知识资源，模拟政府对科普内容创作的定向扶持。
- 资源提升量：0.008/步
- 目标智能体：内容生成智能体

### 实验二：跨域协作激励策略
增加跨领域协作的激励系数，模拟鼓励不同科普领域AI智能体之间的交叉合作。
- 跨域协作激励系数：0.5

### 实验三：质量反馈闭环策略
引入质量评估反馈机制：高质量智能体获得正向激励，低质量智能体受到约束。
- 高质量奖励：0.005
- 低质量惩罚：0.003
- 高质量阈值：0.6
- 低质量阈值：0.3

## 核心演化方程

### 知识更新
```
K_i(t+1) = K_i(t) + α_K * Σ w_ij * max(K_j - K_i, 0) + Δ_R * I(target) + η_K * ε_i
```

### 创新能力更新
```
C_i(t+1) = C_i(t) + β_C * D_i * (1 - C_i) - δ_C * (1 - D_i) * C_i
```

### 内容质量计算
```
Q_i(t) = w_K * K_i + w_C * C_i + w_R * R_i + γ * Collab_i
```

### 系统创新指数
```
Innovation(t) = (1/N) * Σ C_i * D_i * K_i
```

## 实验结果

| 实验名称 | 最终质量 | 最终知识 | 最终创新 | 创新指数 | 基尼系数 |
|---------|---------|---------|---------|---------|---------|
| 基线实验 | 0.6887 | 0.8596 | 0.6940 | 0.3893 | 0.0682 |
| 实验一(资源定向投入) | 0.7522 | 0.9528 | 0.7035 | 0.4289 | 0.0502 |
| 实验二(跨域协作激励) | 0.7209 | 0.8723 | 0.6928 | 0.3956 | 0.0668 |
| 实验三(质量反馈闭环) | 0.8018 | 0.9946 | 0.7973 | 0.5123 | 0.0388 |

## 安装与运行

### 环境要求
- Python 3.8+
- NumPy
- Matplotlib
- NetworkX

### 安装依赖
```bash
pip install numpy matplotlib networkx
```

### 运行仿真
```bash
python simulation_platform.py
```

运行后将自动生成可视化结果，保存在 `simulation_results/` 目录下。

## 核心发现

1. **质量反馈闭环策略效果最佳**：实验三在内容质量、知识水平、创新能力和创新指数四个指标上均表现最优，基尼系数最低（质量分布最均衡）。

2. **资源定向投入次之**：实验一在提升知识水平和内容质量方面效果显著，但创新能力提升有限。

3. **跨域协作激励效果有限**：实验二各项指标提升相对较小，可能需要更强的激励力度。
